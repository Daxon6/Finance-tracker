# Finance Tracker

This project was developed as part of an internship assignment. The goal was to build a finance tracking system with a structured backend architecture and full CRUD functionality.

## Tech Stack

- Backend: Java / Spring Boot
- Database: MySQL
- Build Tool: Maven
- API: RESTful services

---

## My Contributions

My work focused on backend development, specifically designing and implementing the **Employee module** and related system architecture:

- Designed and implemented the complete **Employee entity architecture**
  - Entity
  - Repository
  - Service layer
  - Controller layer

- Developed full **CRUD operations** for Employee management:
  - Create
  - Read
  - Update
  - Delete

- Implemented a **Global Exception Handler** for:
  - Validation of employee fields
  - Handling errors in CRUD operations
  - Providing consistent API error responses

- Built functionality for **bulk export of employee data to CSV format**
